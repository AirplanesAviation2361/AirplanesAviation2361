# Odysha Airlines GeoFS Liveries

Welcome to the official **Odysha Airlines GeoFS liveries repository**!  
This repository contains all liveries for Odysha Airlines planes to use in [GeoFS](https://www.geo-fs.com/).

---

## Airline Information

- **Name:** Odysha Airlines  
- **ICAO Code:** ODY  
- **Callsign:** ODYSHA  
- **Website:** [https://www.odysha-airlines.com](https://www.odysha-airlines.com)  

---

## Livery Overview

Odysha Airlines currently has **3 liveries** for the Airbus A320:

| Livery Name           | Description                                           | Thumbnail |
|-----------------------|-------------------------------------------------------|-----------|
| Normal Livery         | Standard Odysha Airlines livery for everyday flights | ![Normal](thumbnails/normal.png) |
| Retro Livery          | Classic retro design inspired by the original look   | ![Retro](thumbnails/retro.png) |
| India’s 79 Years Livery | Special livery celebrating 79 years of India’s independence | ![India79](thumbnails/india79.png) |
